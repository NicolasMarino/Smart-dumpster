### Prerequisites
Hardware
```
Arduino Uno
At least one: SG90 9G Micro Servo Motor
```
Software:
```
Node.js 
Python  
Arduino IDE
```
### Installing

To start the server
```
cd sd
npm install
npm start
```
To start the script.py
```
cd sd
pip install pyserial
python script.py
```
