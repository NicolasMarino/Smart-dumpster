# Smart dumpster

## Getting Started

### Prerequisites
Hardware
```
Arduino Uno
SG90 9G Micro Servo Motor
```
Software:
```
node.js 
python  
arduino IDE
```
### Installing

To start the server
```
cd sd
npm install
npm start
```
To start the script.py
```
cd sd
pip install pyserial
python script.py
```
